## 概要

- React/TypeScript で、リアルタイムの時計表示を行うアプリケーションです

- 加えてオプションで、ユーザー情報をランダムで生成してくれる外部APIにリクエストする機能があります

## 内容

- 現在時刻の表示

  - 選択したロケールによって、表示される時刻が変わります

- 外部APIを叩いてランダム生成されたユーザ―情報を表示

## 使用技術

- TypeScript

- React

- Next.js

- Material UI (v5)

- React-Select

- useSWR

- hygen

- eslint

- prettier
